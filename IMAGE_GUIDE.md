## 🎨 필요한 이미지 목록

### 1. Open Graph 이미지 (`og-image.png`)
**용도:** Facebook, LinkedIn, 카카오톡 등 소셜 미디어 공유 시 표시

**사양:**
- 크기: **1200 x 630px** (필수)
- 포맷: PNG 또는 JPG
- 파일명: `og-image.png`
- 최대 용량: 8MB 이하 (권장: 300KB 이하)

**디자인 가이드:**
- **텍스트 안전 영역**: 중앙 1200x600px
- **타이틀**: "Dodge Master" (대형, 굵게)
- **서브타이틀**: "Test Your Reflexes!" 또는 "피하기 게임"
- **게임 스크린샷**: 실제 게임 플레이 장면
- **브랜드 컬러**: 파란색(#4a90e2), 초록색(#28a745)
- **CTA**: "Play Now" 버튼 또는 게임 요소

**참고 이미지 구성:**

### 3. Favicon 이미지들

#### 3a. `favicon-32x32.png`
**용도:** 브라우저 탭 아이콘 (표준)

**사양:**
- 크기: **32 x 32px**
- 포맷: PNG
- 투명 배경 권장

**디자인:**
- 단순하고 인식 가능한 아이콘
- 예: 게임 캐릭터(노란 원), 총알 모양, 또는 "D" 로고

#### 3b. `favicon-16x16.png`
**용도:** 브라우저 탭 아이콘 (작은 크기)

**사양:**
- 크기: **16 x 16px**
- 포맷: PNG
- 투명 배경 권장

**디자인:**
- 32x32 아이콘을 단순화
- 디테일 최소화 (픽셀 단위)

#### 3c. `apple-touch-icon.png`
**용도:** iOS 홈 화면 아이콘

**사양:**
- 크기: **180 x 180px**
- 포맷: PNG
- 모서리 둥글게 자동 처리됨 (직각으로 디자인)
- 투명 배경 비권장 (흰색/단색 배경)

**디자인:**
- 앱 아이콘처럼 보이도록 디자인
- 여백 10px 권장
- 고해상도 (Retina Display 대응)

## 📝 빠른 이미지 생성 방법

### 방법 1: 게임 스크린샷 활용
1. 게임 플레이 중 F12 개발자 도구
2. 캔버스 우클릭 → 이미지로 저장
3. 이미지 편집기로 텍스트 추가
4. 크기 조정 (1200x630px)

### 방법 2: Canva 템플릿
1. Canva 가입 (무료)
2. "Facebook Post" 또는 "Twitter Post" 템플릿 검색
3. "Game" 키워드로 필터링
4. 텍스트 수정: "Dodge Master"
5. 다운로드 (PNG 형식)

### 방법 3: 온라인 Favicon 생성기
1. **Favicon.io** (https://favicon.io)
   - 텍스트 → Favicon 변환
   - 이모지 → Favicon 변환
   - 이미지 → Favicon 변환 (여러 크기 자동 생성)

2. **RealFaviconGenerator** (https://realfavicongenerator.net)
   - 업로드 1개 → 모든 크기 자동 생성
   - iOS, Android 아이콘 포함


### 테스트 방법
1. **Open Graph 테스트**
   - https://developers.facebook.com/tools/debug/
   - URL 입력 → "Fetch new information"
   - 이미지 미리보기 확인

2. **Twitter Card 테스트**
   - https://cards-dev.twitter.com/validator
   - URL 입력 → "Preview card"
   - 이미지가 제대로 표시되는지 확인

3. **Favicon 테스트**
   - 브라우저에서 사이트 열기
   - 탭에 아이콘이 표시되는지 확인
   - 북마크 추가 후 아이콘 확인

## 📊 이미지 최적화 팁

### 파일 크기 줄이기
1. **TinyPNG** (https://tinypng.com)
   - PNG 압축 (화질 손실 거의 없음)
   - 최대 5MB → 300KB 이하로 압축

2. **ImageOptim** (macOS 무료 앱)
   - 드래그 앤 드롭으로 자동 최적화

3. **Squoosh** (https://squoosh.app)
   - Google 제공 온라인 이미지 압축 도구

### SEO를 위한 이미지 alt 텍스트
```html
<meta property="og:image:alt" content="Dodge Master game screenshot - test your reflexes by dodging projectiles">
```

## 📎 부록: 빠른 시작 스크립트

이미지 파일을 준비했다면, 아래 명령어로 한 번에 최적화:

```bash
# TinyPNG CLI 설치 (Node.js 필요)
npm install -g tinypng-cli

# 모든 이미지 최적화
tinypng og-image.png twitter-image.png favicon-*.png apple-touch-icon.png

# 파일 크기 확인
ls -lh *.png
```

**목표 파일 크기:**
- og-image.png: 100-300KB
- twitter-image.png: 100-300KB
- favicon-32x32.png: 1-5KB
- favicon-16x16.png: 1-3KB
- apple-touch-icon.png: 10-30KB
